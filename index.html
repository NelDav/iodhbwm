iodhbwm - Die neue LaTeX Klasse

Das Herunterladen von LaTeX Templates für wissenschaftliche Arbeiten ist zu Ende. Nie wieder sich durch einzelne Dateien wühlen. Nie wieder nach dem passenden Befehl suchen.

iodhbwm wird all diese Probleme lösen.

\documentclass[
    load-dhbw-templates,
    auto-intro-pages,
    add-tocs-to-toc,
    language = english,
    mainlanguage = ngerman
]{iodhbwm}
\usepackage[T1]{fontenc}

\dhbwsetup{%
    author            = Max Mustermann,
    thesis type       = PA,
    thesis title      = Einführungsbeispiel mit empfohlenen Einstellungen,
    student id        = 12345,
    location          = Musterstadt,
    institute         = Musterwerke GmbH,
    course/id         = Txxxx,
    supervisor        = Felix Faltin,
    processing period = {01.01.17 -- 31.01.17},
}

\begin{document}
    \chapter{Hello World}
      So schnell hast du noch nie eine wissenschaftliche Arbeit geschrieben.
\end{document}
